Pour éxecuter il faut faire un make ensuite ./program nom_fichier avec nom_fichier étant un fichier texte que l'on
va pouvoir lire.
D'après nos tests le programme respecte toutes les demandes du sujet. Nous avons mis ~ comme nom pour racine que l'on pourra
 changer en vide mais cela a surtout servi lors des vérifications pour nos tests.
Dans notre projet on a veillé à éviter toutes fuites mémoires.
